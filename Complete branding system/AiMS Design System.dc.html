Rules
Never recolor, distort, or place over busy imagery. Always “AiMS” — capital A-M-S, lowercase i. The circle-i glyph is decorative only and never replaces the logo.
Organic shapes
Soft sky-blue rounded shapes add depth behind heroes, empty states, and report covers. Keep them decorative — never behind dense data. Asset: shapes-sky.png
Accent line
Chartreuse tapered underline beneath key headings. Standard rules are 1px with rounded ends.
